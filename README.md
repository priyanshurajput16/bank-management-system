# 🏦 Bank Management System

A banking application built using **Python** and **Streamlit** that enables users to perform basic banking operations through a simple and interactive web interface.

🔗 **Live Demo:** https://bank-management-system-app.streamlit.app/

## ✨ Features

* Create a new account
* Deposit and withdraw money
* Secure account authentication using PIN
* View and update account details
* Delete accounts
* Automatic account number generation

## 🛠️ Tech Stack

* Python
* Streamlit
* JSON
* Object-Oriented Programming (OOP)

## 📥 Clone Repository

```bash
git clone https://github.com/priyanshurajput16/bank-management-system.git
cd bank-management-system
```

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
